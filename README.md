# Eatlyx - Food Delivery App

[![Netlify Status](https://api.netlify.com/api/v1/badges/40664ea6-3719-432d-8ca9-80a655191103/deploy-status)](https://app.netlify.com/projects/eatlyx/deploys)

This is a web application for ordering food from a restaurant. Users can browse the menu, add items to their cart, and place an order.

## Features

- User authentication (login/logout)
- Browse a list of available meals
- Add and remove items from the shopping cart
- Place an order
- View order history

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/veinhorn/reactjs-homeworks.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Technologies Used

- React
- Vite
- Redux Toolkit
- React Router
- Firebase (for authentication)
